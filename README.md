**-INSTALLATION-**

*1*
```bash
git clone https://github.com/ZKRV02/Github_user_activity_cli
```
*2*
```bash
cd Github_user_activity_cli
```
*3*
```bash
pip install .
```
or

*( `pip install -e .` if you want to edit the code)*


**-USAGE-**
just enter github-activity and then enter a valid github name

github-activity
